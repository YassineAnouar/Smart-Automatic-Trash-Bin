# Smart-Automatic-Trash-Bin
An Arduino-based smart trash bin that opens automatically using an ultrasonic sensor and a servo motor.

## Components
- Arduino Uno
- Ultrasonic Sensor
- Servo Motor

## Features
- Touchless lid opening
- Automatic detection
- Simple low
